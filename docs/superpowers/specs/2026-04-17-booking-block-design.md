# Блок записи — Agent Permanent
**Дата:** 2026-04-17  
**Статус:** На согласовании

---

## Цель

Добавить на сайт удобный блок записи который снижает барьер для клиента до минимума. Клиент должен оставить контакт за 5 секунд. Менеджер сам уточнит детали при звонке.

---

## Страница Студии

### Три варианта записи (карточки)

**Карточка 1 — Заявка на перезвон**
- Поля: Имя, Телефон
- Кнопка: "Перезвоним вам"
- При отправке: уведомление в Telegram с хэштегом `#студия`

**Карточка 2 — Онлайн-запись**
- Кнопка открывает виджет YClients
- Клиент сам выбирает дату и время

**Карточка 3 — Написать самому**
- Кнопки: Telegram, Макс, VK

---

## Страница Курса (Академия)

### Две кнопки по странице

- **В начале страницы:** "Узнать дату и стоимость" → открывает попап
- **В конце страницы** (после программы и отзывов): "Записаться на курс" → тот же попап

### Попап академии

- Поля: Имя, Телефон
- Кнопка: "Узнать дату и стоимость →"
- Разделитель: "или напишите сами:"
- Кнопки мессенджеров: Telegram, Макс, VK
- При отправке: уведомление в Telegram с хэштегом курса

---

## Telegram-оповещения

Все заявки с форм приходят в Telegram через бота. Формат сообщения:

```
📩 Новая заявка #студия

Имя: [имя]
Телефон: [телефон]
──────────────────
Источник: Страница студии
```

### Хэштеги по страницам

| Страница | Хэштег |
|---|---|
| Студия | `#студия` |
| Курс ПМ | `#академия #пм` |
| Курс бровиста | `#академия #брови` |
| Курс ламимейкера | `#академия #ламинирование` |

## Технические требования

- Telegram Bot API — бесплатно, мгновенные уведомления
- YClients — существующий виджет, встраивается кнопкой
- Попап — нативный JS, без сторонних библиотек
- Плавающие кнопки мессенджеров на мобильном (фиксированные, всегда видны)

# Runbook — регистрация в Яндекс.Вебмастере

**Цель:** научить Яндекс целенаправленно индексировать сайт + получать статистику запросов и позиций.

**Время:** ~30 минут разово.

**Аккаунт:** тот же Yandex-аккаунт, что для Метрики (счётчик 109018359).

---

## Шаг 1. Добавить сайт

1. Открыть https://webmaster.yandex.ru/
2. Войти Яндекс-аккаунтом (тот же, что для Метрики).
3. Кнопка **«+ Добавить сайт»**.
4. Ввести: `https://agentpermanent.ru`
5. **Continue**.

## Шаг 2. Подтвердить права

Самый простой способ — **через мета-тег**.

1. Я.Вебмастер выдаст строку вида:
   ```
   <meta name="yandex-verification" content="abcdef1234567890">
   ```
2. **Скопировать `content`-значение** (хеш) и **прислать его Игорю-разработчику** (Claude). Я добавлю мета-тег во все 9 production-страниц через PAGES.py + новый скрипт inject_verifications.py.
3. После деплоя нажать в Я.Вебмастере **«Проверить»**.

**Альтернатива (если не хочется ждать деплоя):** загрузить файл с уникальным именем в корень сайта (Я.Вебмастер выдаст). Я могу создать этот файл сразу.

## Шаг 3. Добавить sitemap

1. После верификации — раздел **«Индексирование → Файлы Sitemap»**.
2. Кнопка **«Добавить файл Sitemap»**.
3. Ввести: `https://agentpermanent.ru/sitemap.xml`
4. **Добавить**.

В течение 1–7 дней Яндекс пройдётся по всем URL и начнёт индексировать.

## Шаг 4. Связать с Метрикой

1. Раздел **«Привязка к Я.Метрике»**.
2. Выбрать счётчик **109018359** (Agent Permanent).
3. **Привязать**.

Это даст возможность видеть в Метрике поисковые запросы (какие фразы привели на сайт) и в Вебмастере — поведенческие.

## Шаг 5. Регион

Раздел **«Информация о сайте → Региональность»**:

- Главный регион: **Москва**.
- Это критично для локальной выдачи студии.

## Шаг 6. Турбо-страницы (опционально)

Можно подключить Яндекс.Турбо позже, если будет блог — это даёт мгновенно загружающиеся версии для мобильных. Сейчас не критично.

---

## Что мониторить раз в неделю

В Я.Вебмастере:

- **Индексирование** — все ли страницы из sitemap проиндексированы.
- **Поисковые запросы** — по каким фразам нас уже находят (тут возникнет семядро для Этапа 2).
- **Ошибки** — 4xx, 5xx, дубли.
- **Внешние ссылки** — кто на нас сослался (полезно для SMM-стратегии).
